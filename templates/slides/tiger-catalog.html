TIGER Catalog
http://catalog.coloradocollege.edu

An ongoing experiment for replacing Colorado College's legacy ILS, the first iteration of the TIGER Catalog Minimum Viable Product was released in 2014 using Aaron Schmidt's design he outlined in a blog post. The first release of this TIGER catalog illustrated the front-end viability of using Flask - a Python web microframework - for creating dynamic and lightweight web applications.
